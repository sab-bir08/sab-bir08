## 📂 Featured Projects

> A selection of professional analytics work. Source data and code are confidential, so these are skills-focused summaries of what I built.

### 📌 Automated Partner Performance Reporting Pipeline · Python
End-to-end Python pipeline that auto-detects monthly Excel datagrid exports, computes residual, turnover, and volume metrics per merchant, compiles them into a single time-series dataset, and generates an interactive HTML report. Handles 200+ partners across 13 months with rolling baseline-vs-recent comparison windows, result caching, and automatic riser/faller detection.

`Python` · `Pandas` · `openpyxl` · `Chart.js`

### 📌 Power BI Partner Performance Report — Row-Level Security · Power BI
Designed a Power BI report with DAX-based Row-Level Security so each regional sales manager sees only their own team's accounts, with interactive slicers for partner and territory analysis. Automated dataset refreshes on a Mon–Fri schedule via Power Automate.

`Power BI` · `DAX` · `RLS` · `Power Automate`

### 📌 Mobile Data Usage Distribution Analysis · Python
Statistical deep-dive into SIM data-usage patterns. Identified a heavy-tailed, bimodal distribution and fit a two-component log-normal mixture model to separate light and heavy user populations — surfacing pool-limit risk that a simple average completely hid.

`Python` · `NumPy` · `Matplotlib` · `Statistical Modelling`

### 📌 Monthly Logistics Performance Report · Reporting
Recurring monthly report that turns raw operational data into an executive-ready deck — daily dispatch trends, back-office throughput tables, and category breakdowns with clean, consistent visual design.

`PowerPoint` · `Excel` · `Data Visualisation`
